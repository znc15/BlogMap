# 日志规范

> 本项目使用简单的控制台日志，带时间戳和级别前缀。

---

## 日志工具

位于 `utils/logger.js`，封装 `console.log` / `console.error`：

```js
logger.info('消息', 可选数据);
logger.warn('警告', 可选错误);
logger.error('描述', 错误对象);
```

输出格式：`[YYYY-MM-DD HH:mm:ss] [级别] 消息`

## 日志级别

| 级别 | 使用场景 |
|------|---------|
| INFO | 启动/关闭、数据库连接、GeoJSON 缓存状态、请求摘要 |
| WARN | 非致命异常（GeoJSON 拉取失败回退缓存） |
| ERROR | 数据库操作失败、文件系统错误、未处理异常 |

## 记录内容

- 数据库操作异常：记录 SQL 语句和参数
- 文件上传失败：记录文件名和失败原因
- API 错误：记录路由、错误消息

## 不记录

- 用户密码（即使是哈希）
- 完整的文件内容
- 无关的调试细节
